Crear una Clase e instanciarla

pagina de inicio


Paso 1, Crear una clase

Para crear una clase, ya creado nuestro proyecto, en el paquete que necesitemos crear la clase presionamos clic derecho y seleccionamos la opcion Nuevo y seleccionamos la opcion Java Class. Le tendremos que asignar un nombre a la clase creada, relacionado con lo que necesitemos hacer en esa clase. Se crea la clase con el nombre del paquete al que pertenece y se crea el "public class nombre_clase;" automaticamente.



Paso 2, Creando Atributos en una clase

Dentro de la clase creada, necesitamos crear los atributos que vayamos a necesitar. Para crear los atributos se debe seguir la siguiente sintaxis: "tipo_dato nombre_variable;" en este ejemplo utilizamos dos atributos que son enteros, para ello se utiliza la palabra reservada "int" y la letra "a" representa el nombre de la variable. Para utilizar las variables debemos inicializarla, en este caso la inicializamos en a=0.


Paso 3, Crear un constructor

Cuando en una clase no especificamos ningún tipo de constructor, el compilador añade uno público por omisión sin parámetros, el cual NO hace nada. Un constructor, tiene el mismo nombre de la clase a la cual pertenece. No puede ser Heredado. No retorna ningún valor (Ni void), por lo cual no debe especificarse ningún tipo de dato. Debe declararse como public, sólo en casos realmente extraordinarios será de otro tipo.



Paso 4, Crear un método

Los métodos en Java son apartados en los que se pueden realizar varios procedimientos, en una buena practica se recomienda utilizar métodos para evitar la sobreescritura de código y poder utilizar el método varias veces, sólo mandandolo a llamar. Para crear un método, se debe escoger el tipo de privacidad, tipo de retorno y el nombre del método. En este ejemplo, el método es publico y se utiliza la palabra reservada "public", este metodo no retorna ningun dato, por eso se utiliza la palabra reservada "void". Y en este caso el nombre del método es "Suma". Los paréntesis indican los parametros que recibe el método, en este caso no recibe ningun parámetro. En este caso le indicamos que sume los números "a" y "b" con los números dados.



Paso 5, Crear una instancia de una clase

En la clase principal en la que tenemos el método main, creamos la instancia de la clase a la que queremos llamar. Necesitamos crear un objeto de la clase que necesitamos llamar, esto se hace escribiendo el nombre de la clase (en este caso Ejemplo) y le asignamos un nombre al objeto (ejemplo) lo igualamos a crear un nuevo objeto escribiendo "new nombre_clase". Con esto ya tenemos nuestro objeto de la clase Ejemplo en nuestra clase Tutorial.



Paso 6, Acceder a los atributos y métodos de una clase

Para acceder a los atributos y a los métodos de una clase, se debe colocar el nombre que le dimos al objeto, en este caso (ejemplo) seguido del nombre se coloca un punto y nos arroja la lista de metodos o atributos que tiene la clase, se selecciona el metodo o atributo que se quiere llamar. Si no hubieramos hecho el constructor vacio de la clase, no podríamos instanciarla.

Paso 7, Seleccionar el método que necesitamos

Ya que accedimos a los atributos y a los metodos que tiene nuestra clase Ejemplo, seleccionamos lo que en este caso necesitamos que se muestre, que es el método suma(). Al seleccionarlo accedemos a los procedimientos que realiza este metodo, que es sumar los numeros a y b y mostrar el resultado de la suma.

Paso 8, Correr el programa

Cuando corremos el programa nos da como resultado la suma entre ambos números, lo corremos desde la clase principal que es la que contiene el método main, llamando al metodo de la otra clase que es la clase Ejemplo. Como podemos observar, nos da un resultado correcto.
